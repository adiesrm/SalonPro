# SalonPro

A modern React Native + Expo salon booking application.

## Features

- Firebase Authentication
- Appointment Booking
- Barber Selection
- Booking History
- User Profile
- Secure Logout
- Firestore Backend

## Tech Stack

- React Native
- Expo
- TypeScript
- Firebase Authentication
- Cloud Firestore
- React Navigation
